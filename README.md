## Escenario
**Escenario D – Estadísticas de Resultados Deportivos**
Análisis estadístico de un torneo de fútbol con equipos argentinos.

## Dataset
Dataset generado para el TP ubicado en `/datos/resultados_torneo.csv`.
Contiene: fecha, equipo_local, equipo_visitante, goles_local, goles_visitante.

## Estructura del repositorio
- `/datos` — Dataset CSV del torneo
- `/scripts` — Script Python de análisis
- `/resultados` — Resumen del torneo en texto
- `README.md` — Documentación del proyecto
- `.gitignore` — Exclusión de archivos sensibles

## Cómo ejecutar
1. Clonar el repositorio en Google Colab
2. Ejecutar: `!python scripts/analisis_deportivo.py`
3. Los resultados se guardan automáticamente en `/resultados`
